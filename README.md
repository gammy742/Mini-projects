# Grade System (C++)

A simple C++ project demonstrating **two programming approaches** for calculating student grades:

* Object-Oriented Programming (OOP)
* Procedural Programming (Function-based)

This project was created for learning and comparing programming paradigms.

---

## Features

* Input student name and score
* Automatic grade calculation
* Score validation (0–100)
* Two implementation styles:

  * OOP using Class
  * Function-based programming

---

## Technologies Used

* C++
* Object-Oriented Programming (Encapsulation, Constructor)
* Standard Library (`iostream`, `string`)

---

---

## How to Compile

### OOP Version

```bash
g++ oop_version.cpp -o oop
```

Run:

```bash
./oop
```

---

### Function Version

```bash
g++ function_version.cpp -o func
```

Run:

```bash
./func
```
