# Chicken Company Job Application System (C++)

A console-based job application system built using Object-Oriented Programming in C++.

## Features
- Apply for multiple job positions
- Position-specific validation rules
- Polymorphism using virtual functions
- Input validation system

## Concepts Used
- OOP (Encapsulation, Inheritance, Polymorphism)
- Enum
- Pointer to Base Class
- Runtime Polymorphism

## Positions
- Graphic Designer
- Marketing
- Human Resource
- Accounting

```bash
g++ main.cpp -o app
